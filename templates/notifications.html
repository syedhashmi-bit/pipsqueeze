How to get a Discord Webhook URL:
1. Open your Discord server → right-click the channel → Edit Channel
2. Go to Integrations → Webhooks → New Webhook
3. Copy the webhook URL and paste it above.

Telegram Bot
🤖
#
How to set up a Telegram bot:
1. Open Telegram → search @BotFather → send /newbot
2. Follow the prompts — copy the bot token above
3. Add the bot to your group or start a DM with it
4. Get your Chat ID: send a message, then visit
   https://api.telegram.org/bot<TOKEN>/getUpdates
5. Look for "chat":{"id": ...} — that number is your Chat ID.
Use a negative number for group chats (e.g. -1001234567890).
